# E Commerce App

## Introduction
This is an e-commerce app. It has features like adding, removing, and checking out products.

## Features
- Add products to cart
- Remove products from cart
- Check out products
- Search products
- Lazy loading of pages
- Responsive design

## Installation
    ```
    pnpm install
    ```

    ```
    pnpm run dev
    ```
